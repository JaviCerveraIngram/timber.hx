class LoggerConfig

@:directlyUsed

Creation time configuration for a Logger.

Methods

getFilename():String

Returns:

The filename of the log. This must not include file extension, since this will be indicated to each LoggerWriter by its associated LoggerFormatter.

getHandlers():Array<LoggerHandler>

Returns:

The array of LoggerHandler objects that will be passed to the Logger.

getLevel():Int

Returns:

The default logging level. Only messages logged with this level or below will be output. These will be passed to each LoggerWriter who has not specified a different logging level when the Logger is created.

getPath():String

Returns:

The path where the log file will be created. This is only the directory name, filename must be retrieved with getFilename.
